Willkommen auf den Seiten von Jens Stahl


Dipl.-Wirt.-Inf. (FH) Jens Stahl
Auf diesen Seiten finden Sie Informationen über meine Tätigkeit als freiberuflicher Software-Entwickler im Bereich Java Enterprise Edition. Seit über zehn Jahren entwickle ich komplexe Unternehmens-Software und habe mich dabei auf Datenbank gestützte Web-Anwendungen spezialisiert.

Projekte habe ich sowohl als externer Berater, als auch als interner Mitarbeiter erfolgreich durchgeführt. Dabei habe ich in verschiedenen Entwicklungsprojekten unterschiedliche Aufgaben wahrgenommen. Dazu gehörten insbesondere die Analyse fachlicher Anforderungen, Datenbank Modellierung, Implementierung, Performance Tuning und Qualitätssicherung.

Meine Arbeitsweise ist transparent, Team-orientiert und engagiert. Die Anforderungen des Kunden stehen für mich im Mittelpunkt. Gerade deshalb bin ich ein Freund von agiler Software Entwicklung, um im stetigen Dialog mit dem Kunden zu stehen und flexibel auf neue oder geänderte Anforderungen reagieren zu können.

Falls Sie Unterstützung für ein Projekt benötigen, würde ich mich über eine Nachricht von Ihnen freuen. Ich lebe in Düsseldorf und habe daher besonderes Interesse an Projekten in näherer Umgebung (Bonn, Köln, Düsseldorf, Duisburg, Ruhrgebiet etc.) Auf meiner Homepage haben Sie die Möglichkeit, einen ersten Eindruck über meine Erfahrungen und Kenntnisse zu gewinnen.
